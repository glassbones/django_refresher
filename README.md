# djangoBook

**Description:**  
A social webapp rendered server side and built exclusively with django.

**Features:**  
Static HTML and CSS  
Secure email authorization via AllAuth  
Validated forms  
Dynamic AJAX DOM Updates  
Styling via Semantic UI  
Image processing via Pillow.py
